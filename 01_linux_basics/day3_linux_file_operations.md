# Day3 Linux File Operation Commands

1.touch
create a new empty file.

2.rm
remove a file.
tip:Be careful when using rm, because deleted files may not go to the recycle bin.

3.cp
copy a file.
``bash
cp notes.txt notes_backup.txt

4.mv
move or rename a file.
```bash
mv old_name.txt new_name.txt  #rename
mv notes.txt backup/         #move
