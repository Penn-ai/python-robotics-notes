## Topics Covered

This repository currently covers the following foundational topics for Python, Linux, Git, and basic robotics programming.

### 1. Linux Basics

I learned basic Linux command line operations for navigating folders, creating files, and managing project directories.

Covered commands include:

- `pwd` — show the current directory
- `ls` — list files and folders
- `cd` — change directory
- `mkdir` — create a new folder
- `touch` — create a new file
- `rm` — remove a file
- `cp` — copy a file or folder
- `mv` — move or rename a file or folder

These commands are important for working in Linux-based robotics development environments.

### 2. Git and GitHub Basics

I learned the basic Git workflow for tracking project changes and uploading work to GitHub.

Covered commands include:

- `git status` — check the current state of the project
- `git add .` — add project changes to the staging area
- `git commit -m "message"` — save a local version of the project
- `git push` — upload local commits to GitHub
- `git log --oneline` — view commit history
- `git remote -v` — check the connected GitHub repository

This workflow helps me manage learning notes, code files, and future robotics projects in a more organized way.

### 4. Simple Robotics Programming Logic

I practiced basic programming logic using simple robot-related examples.

Covered exercises include:

- Defining robot names, speed, time, and distance
- Calculating robot movement distance
- Checking robot battery level
- Using target point lists
- Using `for` loops to simulate multi-point navigation
- Using functions to organize robot actions
- Building a simple robot task manager

Example logic:

- If the battery is enough, the robot continues the task.
- If the battery is low, the robot stops.
- The robot moves through target points one by one.
- Each task consumes part of the battery.
